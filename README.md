# An Discordbot for Davidnet

### Required packages
 - axios
 - discord.js
 - node-fetch


### Built for
 - Ubuntu: Ubuntu 24.04 LTS
 - Node: v18.20.4
 - Npm: v10.7.0


### Installation Ubuntu
 - sudo apt install node
 - sudo apt install npm
 - sudo apt install git

 - git clone https://github.com/dedestem/DNBot/
 - cd DNBot

 - npm install
 - Provide an config

 - node Init.js
 - node index.js
